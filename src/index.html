Syncing the station database downloads and resolves every NPC station and Upwell structure known to adam4eve — including full solar system, constellation, and region lookups via ESI.
This process takes 5+ minutes and runs in the background. You can continue using the app, but network activity will be elevated until it completes.
Updating the SDE downloads a ~130 MB file and requires an app restart to take effect.
